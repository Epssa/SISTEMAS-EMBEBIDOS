# SRS - Especificación de Requerimientos del Sistema

## 1. Introducción
Este documento describe los requerimientos del sistema embebido.

## 2. Alcance
El sistema integrará hardware y software para cumplir funciones específicas.

## 3. Requerimientos Funcionales
- RF1: El sistema deberá capturar datos.
- RF2: El sistema deberá procesar información.
- RF3: El sistema deberá mostrar resultados.

## 4. Requerimientos No Funcionales
- RNF1: El sistema deberá ser eficiente.
- RNF2: El sistema deberá ser confiable.
