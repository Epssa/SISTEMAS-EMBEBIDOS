# SRS - Especificación de Requerimientos del Sistema

## 1. Introducción
Este documento describe los requerimientos del sistema embebido que se desarrollará como parte del proyecto integrador.

## 2. Alcance
El sistema permitirá la integración de hardware y software para realizar tareas específicas de monitoreo y control.

## 3. Requerimientos Funcionales
- RF1: El sistema deberá capturar datos mediante sensores.
- RF2: El sistema deberá procesar la información recibida.
- RF3: El sistema deberá mostrar o enviar los datos procesados.

## 4. Requerimientos No Funcionales
- RNF1: El sistema deberá ser eficiente en el uso de recursos.
- RNF2: El sistema deberá ser confiable durante su operación.
