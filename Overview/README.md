# Proyecto Integrador - Sistemas Embebidos

## Descripción
Este proyecto consiste en el desarrollo de un sistema embebido cuyo propósito es integrar hardware y software para resolver un problema específico mediante el uso de un microcontrolador.

## Objetivo
Diseñar e implementar un sistema embebido funcional que permita demostrar el uso de sensores, procesamiento de datos y control de dispositivos electrónicos.

## Factibilidad
El proyecto es factible debido a que se cuenta con los conocimientos básicos en programación y acceso a los componentes necesarios para su desarrollo. Además, el tiempo asignado permite su implementación y pruebas.
